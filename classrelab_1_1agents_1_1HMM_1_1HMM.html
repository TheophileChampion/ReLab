Detailed Description

Implements a Hidden Markov Model.

This implementation extends upon the paper:

Auto-Encoding Variational Bayes, published in ICLR, 2014.

Authors:

  • Kingma D.
  • Welling M.

More precisely, the HMM extends the VAE framework to sequential data by modeling temporal dependencies using a transition network. Note, this agent takes random actions.
