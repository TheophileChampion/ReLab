Detailed Description

Python wrapper around replay buffer implemented in C++.

The implementation is based on the following papers:

Prioritized experience replay, published on arXiv, 2015.

Authors:

  • Tom Schaul

Learning to predict by the methods of temporal differences, published in Machine learning, 3:9â€“44, 1988.

Authors:

  • Richard S. Sutton

More precisely, the replay buffer supports multistep Q-learning and prioritization of experiences according to their associated loss.

Parameters
capacitythe number of experience the buffer can store
batch_sizethe size of the batch to sample
frame_skipthe number of times each action is repeated in the environment, if None use the configuration
stack_sizethe number of stacked frame in each observation, if None use the configuration
screen_sizethe size of the images used by the agent to learn
p_argsthe prioritization arguments (None for no prioritization) composed of:
  • initial_priority: the maximum experience priority given to new transitions
  • omega: the prioritization exponent
  • omega_is: the important sampling exponent
  • n_children: the maximum number of children each node of the priority-tree can have
m_argsthe multistep arguments (None for no multistep) composed of:
  • n_steps: the number of steps for which rewards are accumulated in multistep Q-learning
  • gamma: the discount factor
